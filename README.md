# 🧮 Task 1 — Simple Calculator

This is a simple calculator built using C++ as part of my CodSoft Internship. It performs basic arithmetic operations based on user input and helps in understanding core programming concepts.

**Features:**
- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division with zero-division handling
- User-friendly interactive input system

**How it works:**
1. User enters two numbers
2. User selects an operation (+, -, *, /)
3. Program displays the result based on the selected operation

**Concepts covered:** Input/output handling, conditional statements (if-else / switch-case), problem-solving logic, basic program structuring, Git & GitHub workflow



## ✅ Task 2 — Console To-Do List Manager

A menu-driven to-do list application that lets users add, view, complete, and remove tasks — with all data persisted to disk, so nothing is lost between sessions.

**Features:**
| Feature | Description |
|---|---
| Add Task | Add a new task with a description |
| View Tasks | See all tasks with status — Pending / Completed |
| Mark Completed | Update a task's status once it's done |
| Remove Task | Delete a task from the list |
| Persistent Storage | Tasks auto-save to `tasks.txt` using file handling |
| Input Validation | Invalid inputs are handled gracefully, no crashes |

**Concepts covered:** File I/O, structs, vectors, modular functions, input validation

## Tech Stack

C++17 · File Handling · STL (vectors, structs) · Modular Programming
