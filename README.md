# Task 1 - Simple Calculator (C++)

## 📌 Project Description
This is a simple calculator built using C++ as part of my CodSoft Internship.  
It performs basic arithmetic operations based on user input and helps in understanding core programming concepts.



## ⚙️ Features
- Addition of two numbers  
- Subtraction of two numbers  
- Multiplication of two numbers  
- Division with zero-division handling  
- User-friendly interactive input system  



## 💻 How it Works
- User enters two numbers  
- User selects an operation (+, -, *, /)  
- Program displays the result based on the selected operation  



## 📚 Key Learnings
- Input and output handling in C++  
- Conditional statements (if-else / switch-case)  
- Problem-solving logic  
- Basic program structuring  
- Git & GitHub workflow  



## 🎯 Outcome
This project helped me strengthen my C++ fundamentals and improved my confidence in logical programming and software development basics.
